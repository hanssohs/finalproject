# Seasonal Eats

Accessible recipe website for college students with 16 budget-friendly recipes organized by season.

**Features:**
- localStorage favorites (persists across sessions)
- Triple filter system (season/time/favorites)
- CSS animations (cards, buttons, hovers)
- WCAG 2.1 AA compliant
- Mobile-responsive, keyboard accessible

**Live:** https://hanssohs.github.io/finalproject/
